# LGCCT
## Preparation
the folder structure should be as follows
```css
LGCCT
├── data
│   └── **.pkl
├── modules
│ 
├── pre_trained_models
│ 
├── src
│ 
└── main.py
```
## Config
the parameters can be set in [`main.py`](main.py)
## Run
```
python ./main.py
```
